# AI Hallucination Risk Detector

The AI Hallucination Risk Detector is a machine learning-based system developed to identify and predict the likelihood of hallucinated responses generated by Large Language Models (LLMs). AI hallucinations occur when a model produces information that appears convincing but is factually incorrect, fabricated, or unsupported by evidence.

This project analyzes various prompt and response characteristics, including prompt complexity, confidence score, citation availability, named entities, numeric claims, and response behavior to determine the risk of hallucination. Multiple machine learning algorithms such as Logistic Regression, Random Forest, and XGBoost were trained and evaluated, with XGBoost achieving the best overall performance.

The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, model evaluation, feature importance analysis, and risk classification. Responses are categorized into Low Risk, Medium Risk, and High Risk groups based on predicted hallucination probabilities, helping organizations identify potentially unreliable AI-generated content before it reaches end users.

## Key Features

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Logistic Regression, Random Forest, and XGBoost models
- Model evaluation using Accuracy, Precision, Recall, F1-Score, ROC Curve, and ROC-AUC
- Feature Importance Analysis for explainability
- Hallucination Risk Scoring
- Risk Classification (Low, Medium, High)
- Business recommendations for hallucination mitigation

The developed model successfully identifies potential hallucinated responses and highlights the key factors contributing to hallucination risk. The solution can be used as a safety layer in AI applications to improve reliability, transparency, and user trust in AI-generated content.
